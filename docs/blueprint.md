# **App Name**: Career Compass

## Core Features:

- Opportunity Aggregation: Fetch and aggregate job, scholarship, internship, fellowship, and grant opportunities from various sources (provided URLs/APIs) into a unified database.
- Intelligent Summarization: Employ a generative AI tool to generate concise summaries and extract key information (eligibility, benefits, deadlines, application link) from lengthy opportunity descriptions to show to the users.
- SEO Optimized Content: Automatically optimize opportunity descriptions with relevant keywords for better search engine visibility.
- Advanced Filtering and Search: Implement keyword search and multi-faceted filtering by category, country/region, and level (undergraduate, graduate, professional).
- Personalized Opportunity Recommendations: Use generative AI tool to analyze user profiles and recommend relevant opportunities, based on the summary information gathered from the different sources. Users can then review those sources directly.
- Email Subscription & Notifications: Allow users to subscribe to email newsletters and receive notifications about new and relevant opportunities.
- Responsive Blog Layout: Design a clean, professional, and mobile-first responsive layout for optimal readability and user experience.

## Style Guidelines:

- Primary color: Sophisticated blue (#3498DB) to convey trust and professionalism.
- Background color: Light gray (#F2F4F7) for a clean and readable interface.
- Accent color: Energetic orange (#F39C12) to highlight calls to action and important information.
- Body font: 'PT Sans' (sans-serif) for clear and readable content.
- Headline font: 'Space Grotesk' (sans-serif) to give a modern and bold touch to the titles.
- Use minimalist icons to represent different categories (jobs, scholarships, etc.).
- Implement a clean and structured layout with clear hierarchy for easy navigation.
- Subtle animations and transitions to enhance user experience and engagement.
